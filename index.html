PROJECTS

Giphy Finder is a web app that uses a Giphy API, allowing you to retrieve gifs straight from Giphy's site! You can select preset common search words, or enter your own search word.

NY Park Viewer is a web app that uses a Mapbox API, allowing you to interact with a map showing the locations of different state parks in New York! You view information about each park and save it to your list of favorite parks.

Audio Visualizer is a web app that displays captivating audio visuals for set of classic tracks! Users can interact with a set of controls to modify how the audio sounds and is visualized.

Widget's Wonder Pets is a desktop extention that allows you to pick a cute pet to accompany you on your screen. Every hour or so, it gives you a reminder to take a break from your screen or an encouraging message.

Solitaire Chess is a single player game with the goal of clearing the board until there is one piece left standing. Every piece has the same rules as in chess, and every move must result in eliminating a piece.
